# Portafolio - Gustavo Hernández

Este repositorio contiene un sitio web estático (HTML/CSS/JS) pensado como hoja de vida profesional para presentar a futuros empleadores.

## Estructura
- `index.html` - Página principal
- `sobre-mi.html` - Información personal y académica
- `proyectos.html` - Descripción de proyectos
- `contacto.html` - Formulario de contacto (demo)
- `css/estilos.css` - Estilos separados
- `js/script.js` - Interacciones básicas
- `img/foto.jpg` - Reemplaza con tu foto real

## Instrucciones para publicar en GitHub Pages
1. Crea un repositorio público (por ejemplo: `portafolio-gustavo`).
2. Sube todos los archivos y carpetas tal como están.
3. En GitHub > Settings > Pages, selecciona la rama `main` y la carpeta `/root`.
4. Espera unos minutos y accede a `https://<tu-usuario>.github.io/portafolio-gustavo/`.

## Notas
- Cambia los textos y enlaces con tu información real.
- Agrega tu foto en `img/foto.jpg`.
- Si deseas, puedo proporcionarte un archivo ZIP ya listo o los comandos Git para subir (te los doy).
